# インフラストラクチャ設計

本ディレクトリにはCloud Run / Jobs / Pub/Sub / Firestore / IAMの設計スペックを格納する。

## ファイル一覧

| ファイル                 | 内容                                   |
| ------------------------ | -------------------------------------- |
| `cloud-run-api.yaml`     | APIサービスの配備パラメータ            |
| `cloud-run-worker.yaml`  | Workerジョブの配備パラメータ           |
| `pubsub.yaml`            | Pub/Subトピック/サブスクリプション定義 |
| `firestore-indexes.json` | Firestore複合インデックス              |
| `iam-policy.md`          | サービスアカウント/IAMロール設計       |
