# Biblicant

論文を読む・探す・メモする体験をひとつにまとめるための Chrome 拡張プロトタイプです。  
研究テーマから重要論文を俯瞰し、PDF を開いて翻訳・構文補助（O/C 可視化）を重ねて読み進められることを目指しています。

## 背景
- 重要論文の全体像と系譜を把握したいが、検索結果が分散しやすい  
- PDF を読みながら翻訳や構文補助を切り替えられると効率が良い  
- メモした論文をすぐ参照できる UI が欲しい

## 主な機能
- **検索 UI（モック）**
  - 研究メモ入力 → 重要論文一覧を表示
  - 系譜（インデント）で依存関係を表示
  - 論文カードから PDF を開く
- **メモ済み論文タブ**
  - メモ済み論文をグリッド表示
  - 年・会議名などの最低限のメタ情報を表示
- **PDF ビューワ**
  - PDF を全画面で表示
  - 画面に映ったページと次ページを優先して翻訳
  - 段落上に翻訳オーバーレイを重ねて表示
  - O/C（目的語・補語）を段落内にハイライトして可視化
  - 翻訳と依存解析を個別に ON/OFF 可能

## 使い方（開発）
```bash
npm install
npm run dev
```

Chrome 拡張として読み込む場合は、`dist` を「パッケージ化されていない拡張機能」として登録してください。

## 構成概要
- `src/`  
  - `components/` … 検索 UI / モーダル / メモグリッド  
  - `pdf/` … PDF ビューワと翻訳・依存解析のロジック  
  - `data/` … モックデータ
- `public/`  
  - `background.js` … 拡張アイコンの挙動  
  - `manifest.json` … Chrome 拡張設定
